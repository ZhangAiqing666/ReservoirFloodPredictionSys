# ReservoirFloodPredictionSys
A Reservoir Flood Prediction System.
Using a B/S architecture, the project integrates Golang's Kratos technology with the frontend Vue.js technology stack, and combines the use of EChart components to achieve full-process visualization of control watersheds and various hydrological data. It builds a comprehensive simulation system covering visualization of control watersheds and rainfall stations, acquisition of real-time data and simulation data generation, flood analysis for database storage, and reservoir flood regulation calculations.
